## 🛠 **API Usage**

### Basic Chat
```bash
curl -X POST "http://localhost:8000/api/v1/assistant/chat" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Create an innovative Web3 marketplace",
    "merchant_id": "user123"
  }'
```

### Enhanced Chat with Context
```bash
curl -X POST "http://localhost:8000/api/v1/assistant/chat" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Design a loyalty program strategy",
    "merchant_id": "user123",
    "assistant_mode": "strategic",
    "prompt_profile": "innovation",

  }'
```
