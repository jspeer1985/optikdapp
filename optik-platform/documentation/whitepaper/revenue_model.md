# 3. Token Price Framework (Supply–Demand Logic)

Token value may be influenced by:

1. Percentage of transactions requiring OptikCoin
2. Staked supply (tokens removed from circulation)
3. Treasury buybacks or fee routing mechanisms
4. Merchant pairing requirements

### Circulating Supply Dynamics

If:

* 60% of tokens are staked
* 20% locked in treasury
* 10% reserved for ecosystem incentives

Only 10% remain actively liquid, increasing scarcity under rising demand.

This creates a structural demand curve tied to platform volume rather than speculation cycles.

# 4. Merchant Pairing Model (Core Differentiator)

Each new merchant onboarding event includes:

• Store NFT or Store Identity Token
• Product NFTs
• Paired OptikCoin liquidity allocation
• Onboarding incentive allocation

### Pairing Logic

When a merchant joins:

1. A predefined OptikCoin allocation is paired to their store identity contract.
2. Liquidity pool support ensures transactional depth.
3. Merchant staking tier is activated.

This creates immediate token utility from onboarding activity.
